## Placement Evaluation Questions

Ask any training institute before enrolling:

| Question | What Good Answer Looks Like |
|---|---|
| How many freshers placed last batch? | Specific number, not a percentage |
| Which companies hired them? | Named companies, not "top MNCs" |
| Timeline to placement? | Specific months, not "usually fast" |
| Can I speak with a recent student? | "Yes, here are their contacts" |
| What does placement support include? | Resume, mock interviews, referrals |
| How long does support continue? | Until placed, not until course ends |
